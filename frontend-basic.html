frontend basics

hello guys! you are here to know the basics of the frontend but before we start that let's have a bit knowladge about our lovely web browsers.

web browsers are the interface that allows the client to communicate with the server, it's the main ingridient in the client server model. web browsers are actually those software that takes the server's address as input and send a request to server and collect the response form the server and serve that to the user.

request is a object that collect information about the user and server as well it's a requset that sends data to server telling it the demand of the client.

response is the object that is sends by the server for the given request. It comprise of the document that consist html + css + javascript files, which help the user to understand the response of the server.

Frontend is basically a file that contains the three files html, css and javascript. As the name clears the front view of the website is the frontend. Lets learn more about the the comprise files.

HTML is the abbrevation for hyper text markup language. It's the root of the document that is presented to the client or user. It has some tags that helps us to create the document.

css (cascading stylesheet) is the language that helps us to beautify the document and to make it more interactive to the user. It's basically a language to manipulate the property of the html element

javascript is a programming language thats runs on the browser. It's used for creating dynamic pages, help in taking input manipulating data and help getting more about the events in the document.
